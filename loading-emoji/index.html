Как самостоятельно выгрузить набор эмодзи

Для работы с наборами используется официальный бот @Stickers. Он доступен в формате мини-приложения (Mini App) и через текстовые команды.

Важно!

После обновления в сентябре 2025 года в мини-приложении могут возникать ошибки. Поэтому для надёжности лучше использовать текстовые команды, особенно при загрузке анимированных и видео-эмодзи.

Далее полная инструкция по созданию и управлению наборами эмодзи для Telegram.

Шаг 1

Создаём новый набор

Способ 1 — через Mini App

  1. Откройте бот @Stickers как мини-приложение.
  2. Перейдите во вкладку «Эмодзи».
  3. Нажмите «Создать набор».
  4. Введите название набора (можно использовать любой язык, пробелы и эмодзи).
  5. Загрузите один файл (эмодзи или стикер) и нажмите «Создать».

Набор создан, но ещё не опубликован.

Способ 2 — через команды

  1. Пишем боту команду: /newemojipack — для создания набора эмодзи.
  2. Выберите формат: статичный, анимированный или видео.
  3. Вводим название набора (любой язык, пробелы, ссылки — разрешены).
  4. Загружаем один элемент → жмём «Создать».

Набор создан, но ещё не опубликован.

Шаг 2

Загружаем элементы

Перед загрузкой важно понимать, какой у вас тип эмодзи — статичные, анимированные или видео. От этого зависит способ выгрузки.

Статичные эмодзи

Проще всего загружать через мини-приложение:

  1. Откройте мини-приложение и выберите созданный набор.
  2. Нажмите «Добавить эмодзи».
  3. Загрузите все файлы по очереди.
  4. Для каждого файла подберите соответствующий стандартный эмодзи.

Анимированные и видео-эмодзи

Через мини-приложение они пока не загружаются (баги). Рабочий способ только один — через команды.

  1. Отправьте команду /addemoji.
  2. Выберите из списка нужный набор.
  3. Отправляйте файлы по одному.
  4. После загрузки каждого файла бот попросит привязать к нему стандартный эмодзи. Отправьте его в ответ.
  5. Повторяйте, пока не загрузите все файлы.
Шаг 3

Публикация набора

Перед публикацией Телеграм попросит ввести второе имя для набора — то самое, по которому будет формироваться ссылка. После этого набор станет доступен по ссылке.

Правила:

Только латинские буквы, цифры и знаки подчёркивания. Пробелы запрещены. Имя должно быть уникальным.

Шаг 4

Редактирование набора

Удалять и менять порядок элементов можно в любой момент.

Способ 1 — через Mini App

Удаление элемента:

нажмите на нужный эмодзи → «Удалить».

Изменение порядка:

зажмите элемент мышью и перетащите на нужное место.

Способ 2 — через команды

Удаление элемента:

отправьте /delemoji, выберите набор и файл для удаления.

Изменение порядка:

отправьте /orderemoji, выберите набор, а затем в открывшемся интерфейсе перетащите элементы в нужном порядке.

Не хотите разбираться в деталях?

Если процесс кажется сложным или у вас просто нет на это времени, вы всегда можете заказать набор у профессионального дизайнера. Он не только создаст качественные эмодзи по вашему ТЗ, но и возьмёт на себя все технические моменты, связанные с подготовкой и загрузкой набора в Telegram.
